# Deprecation + Migration

1. Identify dependents (internal/external) and what breaks.
2. Choose stage timeline (deprecated → migration → disabled → removed).
3. Draft migration guide structure + mapping table.
4. Specify tooling (idempotent + dry-run) or explicit manual steps.
