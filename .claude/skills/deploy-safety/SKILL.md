# Deploy Safety

1. Verify revert commit is identified.
2. Check feature flag exists for new functionality.
3. Confirm DB migrations are backward-compatible.
4. Validate monitoring alerts are configured.
5. Test rollback procedure in staging.
6. Document pre-deploy checklist completion.
